Spotify Search
==================

Application to allow you to search from spotify.

<img src="demo1.gif" alt="screenshot"/>

## Running

Follow those steps:
 - `cd tt-music-search`
 - `yarn install`
 - `yarn start`
 - `rename .env.example to .env`
 - `click on Get Token`
 - `take the token and put it in .env file`
 - `restart the app`
 - open http://localhost:3000

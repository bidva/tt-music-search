Spotify Search
==================

Application to allow you to search from spotify.

<img src="demo.gif" alt="screenshot"/>

## Running

Follow these steps:
 - `cd tt-music-search`
 - `yarn install`
 - `yarn start`
 - `rename .env.example to .env`
 - `click on Get Token`
 - `take the token and put it in .env file`
 - `restart the app`
 - open http://localhost:3000

## Testing
Follow these steps:
- `cd tt-music-search`
- `yarn test`
